# AI Chat Widget Pro

AI чат-виджет для WordPress с интеграцией GigaChat (Yandex GPT) и лидогенерацией через Telegram.

## Возможности

- 🤖 Интеграция с GigaChat API (Yandex GPT)
- 💬 Кастомизируемый системный промпт
- 🎨 Гибкие настройки внешнего вида (цвета, размеры)
- 📱 Полностью адаптивный дизайн
- 🔔 Автоматическая классификация лидов (горячий/тёплый/холодный)
- 📤 Отправка лидов в Telegram
- 🔒 Безопасность (nonce, sanitization, escaping)
- 📊 Сохранение лидов в базу данных

## Установка

1. Загрузите папку `ai-chat-widget-pro` в `/wp-content/plugins/`
2. Активируйте плагин в меню "Плагины" в WordPress
3. Перейдите в "AI Chat Widget" → настройте API ключи

## Использование

Вставьте шорткод в любую страницу или запись:

```
[ai_chat_widget]
```

## Настройка

### Основные настройки

- **API ключ GigaChat** — получите в личном кабинете GigaChat
- **Системный промпт** — инструкция для AI ассистента
- **Имя агента** — отображается рядом с аватаром
- **URL аватара** — изображение ассистента (80x80px рекомендуется)

### Внешний вид

- **Основной цвет** — цвет кнопки и сайдбара
- **Цвет сообщений пользователя** — фон сообщений справа
- **Цвет сообщений AI** — фон сообщений слева
- **Максимальная высота** — высота секции чата
- **Ширина на desktop** — максимальная ширина контейнера

### Telegram интеграция

- **Bot Token** — получите у @BotFather
- **Chat ID** — получите у @userinfobot
- **Отправка лидов** — включить/выключить уведомления

## Классификация лидов

### 🔥 Горячий (высокий приоритет)
Ключевые слова: `купить`, `заказать`, `цена`, `стоимость`, `доставка`, `оплата`, `в корзину`, `куплю`, `хочу купить`

### 🟡 Тёплый (средний приоритет)
Ключевые слова: `интересно`, `подробнее`, `как работает`, `отзывы`, `цена?`, `есть в наличии?`, `скидка?`

### ❄️ Холодный (низкий приоритет)
Приветствия, общие вопросы — не отправляются в Telegram

## Требования

- WordPress 5.8+
- PHP 7.4+
- jQuery (входит в WordPress)

## Безопасность

- Все AJAX запросы защищены nonce
- Пользовательский ввод проходит sanitization
- Вывод данных экранируется (esc_*)
- Подготовленные запросы к БД (prepare)
